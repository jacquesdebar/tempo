# Tempo

A simple, elegant web app to track turn times during board game sessions. Perfect for understanding game pace, improving play speed, or just satisfying your curiosity about who's taking the longest turns!

## Features

- **Track Multiple Players** - Add as many players as needed
- **Flexible Turn Orders** - Sequential, flexible, or round-based gameplay
- **Live Statistics** - See time distribution in real-time
- **Pause Functionality** - Track "admin time" separately  
- **Game History** - Review past games and statistics
- **Mobile Friendly** - Works great on phones and tablets
- **No Installation** - Runs in any modern web browser

## Quick Start

1. Visit the app at [your-vercel-url]
2. Add player names
3. Select your turn order style
4. Hit "Start Game" and begin playing!

## How to Use

### Setup
1. Enter player names (2 minimum)
2. Choose turn order:
   - **Fixed Order**: Players take turns in sequence
   - **Flexible Order**: Any player can take the next turn  
   - **Round-based**: Fixed sequence that repeats each round

### During the Game
- Press **Next Player** when a turn ends
- Use **Pause** for rules discussions or breaks
- Time percentages update live
- Current player and turn time are prominently displayed

### After the Game
- **End Game** shows a complete summary
- View total time, turns, and average turn length per player
- Access game history to review past sessions

## Privacy

All data is stored locally in your browser. No account needed, no data sent to servers.

## Tips

- Name players consistently to track stats across games
- Use pause for setup time, rules explanations, or breaks
- The timer starts automatically when you begin a game
- Previous games are saved automatically
